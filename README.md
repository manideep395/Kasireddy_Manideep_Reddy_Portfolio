# Kasireddy Manideep Reddy — Portfolio

Personal portfolio website showcasing my projects, skills, and experience.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Three.js

## Getting Started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Deployment

Build for production:

```sh
npm run build
```
